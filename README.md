# Cleanistère

Le site web Cleanistère vise à mettre en lumière les actions de ramassage
de déchets dans la zone de Brest.

Le projet est mené en méthode agile sur ce
[projet GitHub](https://github.com/users/Ramdek/projects/1).

## Utilisation du site

Le site est déployé via
[GitHub Pages](https://ramdek.github.io/cleanistere/).

Identifiants de test: `yannlg29:Crode_123!`

A date, il est possible de créer un évènement en placant un marker sur la
carte *(en étant authentifié)*.

## Points d'amélioration

Nous utilisons actuellement le localStorage du navigateur pour ce POC afin de
stocker de la donnée de façon persistante.  
Il faudrait bien évidemment utiliser une base de donnée en temps normal.

Il reste bien évidemment des tickets non terminés par faute de temps.  
Aucune pratique de sécurité n'est mise en place.
